# typescript-10-apps
10 node apps

This is inspired by Michael Kennedy's excellent video course [Python Jumpstart Build 10 apps](https://training.talkpython.fm/courses/explore_python_jumpstart/python-language-jumpstart-building-10-apps)

### Prerequisite

Install ts-node globally

```
npm install -g typescript; npm install -g ts-node
```
### How to Run
1. ```yarn```
2. ```ts-node ./01_hello_world/main.ts```
